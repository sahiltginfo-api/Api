## 🚀 QUICK START

### 1. Get Bot Token
```
→ Open Telegram, search @BotFather
→ Send /newbot
→ Follow instructions
→ Copy token
```

### 2. Get Your User ID
```
→ Search @userinfobot
→ Send /start
→ Copy your ID
```

### 3. Install & Configure
```bash
# Clone/download files
pip install -r requirements.txt

# Create .env file
cat > .env << EOF
BOT_TOKEN=your_token
BOT_OWNER_ID=your_id
API_KEY=Demo
EOF
```

### 4. Run Bot
```bash
python main_premium.py
```

```

→ Send /start
→ Use menu buttons
```

## 🐛 TROUBLESHOOTING

### Bot won't start
```bash
# Check token
grep BOT_TOKEN .env

# Check Python version
python --version  # Should be 3.8+

# Check dependencies
pip list

# See errors
cat bot.log
```

### Database issues
```bash
# Backup and delete
cp osint_bot.db osint_bot.db.backup
rm osint_bot.db

# Restart bot (will recreate)
python main_premium.py
```

### Commands not working
```bash
# Check bot is running
ps aux | grep main_premium.py

# Check logs
tail -f bot.log

# Verify user not banned
/admin → Users → Search
```
